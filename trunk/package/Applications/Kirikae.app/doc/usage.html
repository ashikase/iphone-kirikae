To switch to another running application:


  1. Double-tap the home button to display Kirikae.
  2. Tap the application that you wish to switch to.
    • If you have Backgrounder installed, the current app will be backgrounded; otherwise it will quit.
    • The selected application will then become active.

To quit an active/backgrounded application:


  1. Double-tap the home button to display Kirikae.
  2. Swipe the application that you wish to quit.
  3. Tap the quit button that appears.
